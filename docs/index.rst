Welcome to Django Oscar Easyrec!
=================================

`Oscar`_ is an open-source, domain driven ecommerce framework for Django
and `easyrec`_ is a recommendation system. So naturally `django oscar easyrec`
is a reusable app designed to integrate the two as seamlessly as possible.

.. _`Oscar`: http://oscarcommerce.org
.. _`easyrec`: http://easyrec.org/
.. _`django-oscar`: https://github.com/tangentlabs/django-oscar
